# 物理视图说明

> **最后更新**: 2025-11-23  
> **状态**: 不适用

---

## 为什么没有物理视图？

本项目 `executor-demo` 是一个**业务模块**，而非完整的独立系统。

### 物理视图的适用场景
物理视图（Physical View）通常用于描述：
- 完整系统的部署架构
- 硬件拓扑
- 网络规划
- 服务器资源配置

### 本项目的定位
`executor-demo` 是一个可以被集成到更大系统中的**业务组件/模块**：
- 它是一个 Spring Boot 应用
- 可以作为微服务部署
- 可以作为模块集成到单体应用中
- **部署方式取决于宿主系统的架构**

### 技术依赖
本模块的运行依赖：
- **Redis**: Checkpoint 持久化（可选，也可用 InMemory）
- **Nacos**: 服务发现（可选，支持降级到固定 IP）
- **外部服务**: OB Service, Gateway, Portal（接口集成，非物理部署）

这些依赖的部署方式由**集成本模块的系统**决定，而非本模块本身。
